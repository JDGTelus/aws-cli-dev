Migration to Arch Linux (Historical)

Note: This migration path was attempted but abandoned due to ARM64 compatibility issues. The project now uses Alpine Linux instead.

See alpine-linux-guide.html for current setup.

Why This Was Attempted

Ubuntu 24.04 had Neovim 0.9.5, but nvim-lspconfig requires 0.10+. Arch Linux has the latest packages but doesn't support ARM64 Docker images.

Final Solution

Alpine Linux - Lightweight, multi-platform, with Neovim 0.11+

Base Image: alpine:latest
Neovim: 0.11.1
Platform Support: ARM64 + x86_64
