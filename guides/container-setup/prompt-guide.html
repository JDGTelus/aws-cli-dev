How It Works

The prompt is defined in ~/.zshrc (line 24):

PROMPT='%(?:%{$fg_bold[green]%}aws-cli-env âžœ :%{$fg_bold[red]%}aws-cli-env âžœ )%{$fg[cyan]%}~ %{$reset_color%}$(git_prompt_info)$(aws_region_prompt)'

Breakdown

  1. %(?:...) - Conditional based on last command exit status
    • If successful: Shows green prompt
    • If failed: Shows red prompt
  2. aws-cli-env âžœ - Container name with arrow and space
  3. ~ - Tilde with one space
  4. $(git_prompt_info) - Git branch (from Oh-My-Zsh)
  5. $(aws_region_prompt) - AWS region (custom function)

Customizing the Prompt

Show Current Directory Instead of Tilde

Replace ~ with %c (directory name) or %~ (full path):

# Show directory name
PROMPT='%(?:%{$fg_bold[green]%}aws-cli-env âžœ :%{$fg_bold[red]%}aws-cli-env âžœ )%{$fg[cyan]%}%c %{$reset_color%}$(git_prompt_info)$(aws_region_prompt)'

# Show full path
PROMPT='%(?:%{$fg_bold[green]%}aws-cli-env âžœ :%{$fg_bold[red]%}aws-cli-env âžœ )%{$fg[cyan]%}%~ %{$reset_color%}$(git_prompt_info)$(aws_region_prompt)'

Change Colors

Available colors: black, red, green, yellow, blue, magenta, cyan, white

# Change tilde to yellow
PROMPT='%(?:%{$fg_bold[green]%}aws-cli-env âžœ :%{$fg_bold[red]%}aws-cli-env âžœ )%{$fg[yellow]%}~ %{$reset_color%}$(git_prompt_info)$(aws_region_prompt)'

Zsh Prompt Codes

Code Meaning
%~Current directory (full path with ~ for home)
%cCurrent directory name only
%/Current directory (absolute path)
%nUsername
%mHostname (short)
%TTime (24-hour HH:MM)
%DDate (YY-MM-DD)

Applying Changes

After editing ~/.zshrc:

source ~/.zshrc

Why Theme Files Don't Work

Important: Oh-My-Zsh themes are loaded first, but then .zshrc sets a custom PROMPT variable which overrides the theme.

To customize: Edit line 24 in .zshrc, NOT the theme file.
